AssocLegendre calculate the associated Legendre functions


Inherits from: Object


Calculates the associated Legendre functions of degree l and order m, which are the canonical solutions of the general Legendre equation:


(1-x^2)y" - 2xy' + [l(l+1) - (m^2/(1-x^2))]y = 0.

The associated Legendre functions are used in the construction of the spherical harmonics (see the class SpherHarmonic).



Creation / Class Methods


*new(l, m)

Creates a new instance of AssocLegendre.

l, m - Both an instance of Integer.

*calc(l, m, x)

Calculates the associated Legendre function of degree l and order m for x.

l, m - Both an instance of Integer.

x - An instance of SimpleNumber.

calc(x)

Calculate the associated Legendre function for x. The function is defined for x running from -1 to +1.

Examples




//Create an instance of AssocLegendre with degree 2 and order 1 and calculate for x = 0.5

a = AssocLegendre.new(2, 1);

a.calc(0.5)


//Also possible to calculate directly

AssocLegendre.calc(2, 1, 0.5);


//Graph the function for x running from -1 to 1

a = AssocLegendre.new(2, 1);
