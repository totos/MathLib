SpherHarmonic calculate the spherical harmonics


Inherits from: Object


Calculates the spherical harmonics of degree l and order m.  They form a complete, orthonormal set of functions of theta and phi and are the solutions of the angular part to Laplace's equation.


SpherHarmonic depends on the class AssocLegendre.



Creation / Class Methods


*new(l, m)

Creates a new instance of SpherHarmonic.

l, m - Both an instance of Integer.

*calc(l, m, theta, phi)

Calculates the spherical harmonic of degree l and order m for theta and phi. Note that this method returns a complex valued function of type Complex.

l, m - Both an instance of Integer.

theta, phi - Both an instance of SimpleNumber. Theta represents the polar angle running from 0 to pi and phi represents the azimuthal angle running from 0 to 2pi.


Instance Methods

setL(l)

Set the degree of the spherical harmonic. 

getL(l)

Get the degree of the spherical harmonic.

setM(m)

Set the order of the spherical harmonic.

getM(m)

Get the order of the spherical harmonic.

setLM(l, m)

Set the degree l and order m of the spherical harmonic.

getLM(l, m)

Get the degree l and order m of the spherical harmonic.

calc(theta, phi)

Calculate the spherical harmonic for theta and phi. Theta represents the polar angle running from 0 to pi and phi represents the azimuthal angle running from 0 to 2pi. Note that this method returns a complex valued function of type Complex.

Examples




//Create an instance of SpherHarmonic with degree 3 and order 2 and calculate for theta = 0.25pi and 

//phi = 0.25pi

a = SpherHarmonic.new(3, 2);

a.calc(0.25pi, 0.25pi)


//Also possible to calculate directly

SpherHarmonic.calc(3, 2, 0.25pi, 0.25pi);


//Graph the function for theta fixed at 0.25pi and phi running from 0 to 2pi

a = SpherHarmonic.new(3, 2);

b = 1000.collect({ |i| i = (i / 999) * 2pi; a.calc(0.25pi, i) });

b.real.plot("real", Rect(20, 400, 470, 280));

b.imag.plot("imag", Rect(490, 400, 470, 280));
