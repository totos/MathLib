HydrogenWaveFunction calculate a hydrogen wave function


Inherits from: Object


Calculates the solutions of the Schroedinger equation for a hydrogen atom in spherical coordinates. The radial part is defined by the generalized Laguerre polynomials (see the class GenLaguerre) and the angular part by the spherical harmonics (see the class SpherHarmonic). 


With the solutions are associated a set of three quantum numbers: n - the principal quantum number which takes on positive integer values from 0 up to infinity, l - the azimuthal quantum number which takes on positive integer values from 0 up to n - 1 and m - the magnetic quantum number which takes on integer values from -l up to +l.



Creation / Class Methods


*new(n, l, m)

Creates a new instance of WaveFunction.

n, l, m - All an instance of Integer.

*calc(n, l, m, r, theta, phi)

Calculates a hydrogen wave function for given integer values n, l and m and radius r, polar angle theta and azimuthal angle phi. Note that this method returns a complex valued function of type Complex.

n, l, m - All an instance of Integer.

r, theta, phi - All an instance of SimpleNumber.

*calcAbsSquare(n, l, m, r, theta, phi)

Calculates the absolute square of a hydrogen wave function for given integer values n, l and m and radius r, polar angle theta and azimuthal angle phi. This method returns the probability density of finding the electron at a particular point in space.

n, l, m - All an instance of Integer.

r, theta, phi - All an instance of SimpleNumber.


Instance Methods

setN(n)

Set the principal quantum number n of the wave function. 

getN(n)

Get the principal quantum number n of the wave function.

setL(l)

Set the azimuthal quantum number l of the wave function.

getL(l)

Get the azimuthal quantum number l of the wave function.

setM(m)

Set the magnetic quantum number m of the wave function.

getM(m)

Get the magnetic quantum number m of the wave function.

setNLM(n, l, m)

Set the principal quantum number n, azimuthal quantum number l and the magnetic quantum number m of the wave function.

getNLM(n, l, m)

Get the principal quantum number n, azimuthal quantum number l and the magnetic quantum number m of the wave function.

calc(r, theta, phi)

Calculate the wave function for r, theta and phi. R is the radius taking on positive real values from 0 to infinity, theta is the polar angle ranging from 0 to pi and phi is the azimuthal angle ranging from 0 to 2pi. Note that this method returns a complex valued function of type Complex.

calcAbsSquare(r, theta, phi)

Calculate the absolute square of the wave function for r, theta and phi. R is the radius taking on positive real values from 0 to infinity, theta is the polar angle ranging from 0 to pi and phi is the azimuthal angle ranging from 0 to 2pi. This method returns the probability density of finding the electron at a particular point in space.

Examples




//Create an instance of WaveFunction with n = 2, l = 1, m = 0, r = 8, theta = 0.25pi and phi = 

//0.25pi

a = HydrogenWaveFunction.new(2, 1, 0);

a.calc(8, 0.25pi, 0.25pi)


//Also possible to calculate directly

HydrogenWaveFunction.calc(2, 1, 0, 8, 0.25pi, 0.25pi);


//Graph a spherical ray of the probability density of the wave function for r running from 0 to 8 

//and theta and phi fixed at 0.25pi

a = HydrogenWaveFunction.new(2, 1, 0);

1000.collect({ |i| i = (i / 999) * 8; a.calcAbsSquare(i, 0.25pi, 0.25pi) }).plot


//Graph a cross-section of the probability density of the wave function in 2 dimensions for r 

//running from 0 to 8, theta fixed at 0.5pi and phi running from 0 to 2pi

(

var waveFunc = HydrogenWaveFunction.new(2, 1, 0), res = 500, maxr = 8, dim = 200;


w = GUI.window.new("Hydrogen Wave Function", Rect(100, 400, dim, dim)).front;

w.view.background_(Color.black);


w.drawHook = {

res.do({ arg i; i = i / (res-1);

res.do({ arg j; var radius, theta = 0.5pi, phi, val, xval, yval; j = j / (res-1);

radius = j * maxr;

phi = i * 2pi;

val = waveFunc.calcAbsSquare(radius, theta, phi) * 25e34;

xval = (j * dim) * cos(phi) + (dim * 0.5);

yval = (j * dim) * sin(phi) + (dim * 0.5);

GUI.pen.width = 1;

GUI.pen.strokeColor = Color.hsv(0.7 - (val * 0.22), 1-(val**4), val);

GUI.pen.moveTo(xval@yval);

GUI.pen.lineTo(xval@yval+1);

GUI.pen.draw(2)

})

})

};

w.refresh

)
